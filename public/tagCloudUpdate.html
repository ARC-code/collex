Update on the NINES tag cloud

You've reached this page because the tag cloud you're trying to access no longer exists. Before the 2008 redesign, the NINES Collex interface once offered four different tag clouds, one each for sites, keywords, peers, and dates. After our 2008 redesign, we reduced that number to one: the keywords cloud. Follow this link to see the keywords NINES users have applied to the objects they have collected.
